1. Giải thích các câu lệnh:

git init: khởi tạo

git add <file_name>: thêm 1 file vào vùng staging

git add . : thêm toàn bộ file vào vùng staging

git commit -m “message”: commit file

2. 

a. Tạo 3 file: file1, file2, file3 -> thư mục trên máy local

b. Chạy lệnh: git init -> khởi tạo git quản lý thư mục, nằm ở vùng working directory (file1, file2, file3)

c. Chạy lệnh: git commit -m”init project” -> vùng staging rỗng vì chưa file nào đc add lên staging, nên ko có file nào để commit repository

d. Chạy lệnh: git add file1 -> git thực hiện sao chép file1 từ vùng working directory lên vùng staging

e. Chạy lệnh: git commit -m”add file1”->  tạo ra 1 commit ở trên vùng repository, đưa file1 lên repository và đồng thời xóa file1 ở staging

f. Chạy lệnh: git commit -m”add file” -> vùng staging rỗng, vì chưa có file nào đc add lên staging, nên sẽ khong có file nào được commit lên repository (trên repository hiện tại chỉ có 1 file1, được commit trước đó)
