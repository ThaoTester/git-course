1/

git commit --amend 
-> muốn thay đổi lại commit cuối cùng.

git commit --amend -m”<message>
-> muốn change lại message khi commit

git restore --staged <file>
-> Di chuyển file từ vùng staging về vùng working directory 

git reset HEAD~1
-> Di chuyển file từ vùng repository về vùng working directory
-> Quay lại commit trước đó 1 bước (bỏ commit gần nhất)
